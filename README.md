# clevis

### install
```
npm install -g clevis
```

### install from source
```
git clone https://github.com/austintgriffith/clevis.git
cd clevis
npm install
sudo npm link
```


### in docker
```
npm config set user 0
npm config set unsafe-perm true
```

```
npm install -g clevis
```
